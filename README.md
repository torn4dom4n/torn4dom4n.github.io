# @torn4dom4n/website

Hello humans, my name is Long Nhat Nguyen. This is my website.

## Develop locally

```bash
git clone https://github.com/torn4dom4n/website
cd website
pnpm install
pnpm dev
```
Open your web browser and navigate to http://localhost:5173.
